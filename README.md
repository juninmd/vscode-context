# vscode-context

**Description:** Adds commands to the Windows context.

**Installation:**

1.  Clone the repository: `git clone https://github.com/your-username/vscode-context.git`
2.  Navigate to the repository directory: `cd vscode-context`
3.  Install dependencies (if any): `npm install` (refer to `package.json` for dependencies).

**Usage:**

*   **`vscode-context`:**  This command will execute the following tasks:
    *   `open-file.reg`: Opens the currently open file.
    *   `open-folder.reg`: Opens the currently open folder.
    *   ` renovate.json`:  Runs the `renovate.json` script.
*   **`vscode-context --help`:** Displays available commands and their descriptions.
